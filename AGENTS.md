# AGENTS

## Кодировка
- Все текстовые файлы проекта (`.py`, `.qml`, `.md`, `.txt`, `.csv`) сохранять строго в `UTF-8`.
- Не допускать смешения `UTF-8` и `cp1251` в одном файле.
- При правках русскоязычных строк проверять, что в UI нет «кракозябр».
- Не использовать Unicode-экранирование `\uXXXX` для обычных UI-строк: писать русский текст напрямую в UTF-8.
- После скриптовых/массовых правок обязательно делать пост-проверку измененных файлов на «кракозябры» и сразу исправлять кодировку до `UTF-8` без смешения с другими кодировками.
- Любые скриптовые правки делать только с явным указанием кодировки `UTF-8` при чтении и записи; запрещено использовать команды/инструменты записи с неявной кодировкой (например, `Set-Content` без `-Encoding utf8`), чтобы не допускать повторной порчи текста.
